HTTP Proxy Service Worker

http-proxy.js implements an RFC 7234 and RFC 5861 compliant HTTP proxy suitable for use within a service worker.

Since HTTP has very sophisticated cache-handling features, this makes it possible to implement different caching strategies via minimal wrappers around http-proxy.js. Some examples are below.

Some strategies that parallel the Service Worker Toolbox handlers:

  • Cache First (source)
  • Network First
  • Cache Only
  • Network Only (source)
  • Fastest

Other strategies:

  • Cache Synthetic (source)
  • Pre-cache (source)
  • Stale While Revalidate (source)
