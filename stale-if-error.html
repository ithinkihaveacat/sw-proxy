Stale If Error

This strategy transforms the headers of the following images so that they get a cache-control header of max-age=30, stale-if-error=30.

This has the behavior:

  • image was retrieved less than 30 seconds ago - cached image is returned
  • image was retrieved between than 30 and 60 seconds ago - attempt to retrieve image via network; if network error (e.g. offline), return cached image
  • image was retrieved more than 60 seconds ago, or not in cache - image is retrieved from network
